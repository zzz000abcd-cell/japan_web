10關遊戲流程詳細說明

1

街景獵詞

使用Google Maps街景尋找日文招牌，掃描並記錄文化詞彙。 任務：抄低5個日文詞 → Anki新建卡片

✓ 開Google Maps → 京都祇園（35.0036° N, 135.7750° E）
✓ 用街景慢慢行，搵第一間有日文招牌嘅店舖
✓ 用Google Lens掃描招牌 → 截圖儲存
✓ 成就解鎖：街景獵人 Lv.1（+50分）
2-5

文化詞典大挑戰

每關主題不同，探索不同文化領域：

✨ 第2關：和服文化（振袖、帯、草履）
✨ 第3關：神社文化（お守り、絵馬、手水舎）
✨ 第4關：飲食文化（懐石、湯葉、京野菜）
✨ 第5關：表演藝術（地唄、花柳流、狂言）
✓ 成就解鎖：祇園詞典達人（+250分）
6-9

文化解謎

從日本人角度理解詞彙背後的價值觀，撰寫文化心得

✓ 用電腦Google搜尋學到的3個詞 +「文化」
✓ 從日本人角度理解：詞彙背後的價值觀是什麼？
✓ 遊戲規則：每詞寫1句心得（20字內）
✓ 文化洞察：空間劃分反映日本社會結構
✓ 成就解鎖：文化解碼者（+400分）
10

祇園文化總結王

整合所有心得，撰寫祇園文化總結，成為「祇園文化守護者」

✓ 整合9關心得 → 寫100字祇園文化總結
✓ 格式要求：提煉核心文化價值觀
✓ 文化總結示例：祇園唔單止係觀光區，係日本人「美意識」「禮儀」「時間觀」嘅縮影
✓ 成就解鎖：祇園文化守護者（+500分 + 完成證書）
